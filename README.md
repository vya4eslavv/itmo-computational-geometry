# itmo-computational-geometry

Репозиторий с решениями лабораторных работ по курсу **«Вычислительная геометрия»**.  
В рамках курса рассматриваются аффинные преобразования, однородные координаты, моделирование плоских линий, алгоритмы локализации точки, выпуклые оболочки и задачи геометрического поиска.  

---

## Лабораторная работа 1 — Аффинные преобразования. Однородные координаты

В лабораторной работе рассматриваются преобразования плоских фигур в декартовой системе координат с использованием матриц однородных координат.  
По методичке доступны два варианта:

- **Задание 1.1. Преобразование треугольника**
  - перенос;
  - поворот относительно центра треугольника;
  - осевая симметрия;
  - гомотетия относительно начала координат;
  - гомотетия относительно середины наименьшей стороны;
  - построение композиции преобразований;
  - визуализация исходной фигуры и всех образов.  [oai_citation:2‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

- **Задание 1.2. Преобразование квадрата**
  - построение аффинного преобразования квадрата в параллелограмм;
  - нахождение матрицы преобразования и обратной матрицы;
  - вычисление образов в однородных координатах;
  - проверка корректности преобразования;
  - визуализация результата.  [oai_citation:3‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

### Файлы решения
- `lab1/task_1_1_triangle_transform.ipynb`
- `lab1/task_1_2_square_transform.ipynb`

## Лабораторная работа 2 — Моделирование плоских линий

Лабораторная работа посвящена моделированию линий и кривых на плоскости.  
По методичке предлагаются следующие задания:

- **Задание 2.1. Преобразование квадрики**
  - задание окружности и эллипса;
  - запись уравнения кривой второго порядка в матричной форме;
  - построение аффинного преобразования окружности в эллипс;
  - получение аналитического уравнения образа;
  - сравнение геометрического и аналитического результата.  [oai_citation:4‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

- **Задание 2.2. Кривизна кривой**
  - выбор двух кривых;
  - построение касательного и нормального векторов;
  - вычисление кривизны и радиуса кривизны;
  - построение эволюты, центра кривизны и соприкасающейся окружности.  [oai_citation:5‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

- **Задание 2.3. Кривые Безье**
  - исследование перестановок опорных точек;
  - анализ простых, самопересекающихся и возвратных кривых;
  - применение аффинного преобразования;
  - моделирование контура сложной формы сплайнами и кривыми Безье.  [oai_citation:6‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

- **Задание 2.4. Растровое изображение линий**
  - построение отрезка алгоритмом Брезенхейма;
  - построение отрезка естественным алгоритмом;
  - применение переноса и поворота.  [oai_citation:7‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

## Лабораторная работа 3 — Локализация точки

В лабораторной работе рассматриваются алгоритмы пересечения геометрических объектов и определения положения точки относительно многоугольника.

- **Задание 3.1. Пересечение отрезков**
  - поиск всех пересекающихся неколлинеарных отрезков:
    - по уравнениям прямых;
    - методом косых произведений;
    - методом заметающей прямой;
  - расширение алгоритмов на случай коллинеарных отрезков.  [oai_citation:8‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

- **Задание 3.2. Локализация точки в невыпуклом многоугольнике**
  - определение положения точек:
    - угловым методом;
    - лучевым методом;
  - доработка алгоритмов для случаев, когда несколько вершин лежат на одной горизонтальной прямой.  [oai_citation:9‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

### Файлы решения

- `lab3/task_3_2_point_in_polygon.ipynb`

## Лабораторная работа 4 — Локусы

Лабораторная работа посвящена построению выпуклых оболочек, диаграмм Вороного и триангуляции Делоне.

- **Задание 4.1. Выпуклая оболочка множества**
  - построение выпуклой оболочки множества `G` алгоритмом Грэхема;
  - построение выпуклой оболочки множества `D` алгоритмом Джарвиса;
  - вычисление периметра и площади;
  - поиск пересечения оболочек и внутренних точек.  [oai_citation:10‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

- **Задание 4.2. Близость**
  - построение диаграммы Вороного:
    - алгоритмом Форчана;
    - рекурсивным методом «разделяй и властвуй»;
  - поиск ближайшей пары точек;
  - построение триангуляции Делоне как двойственного графа.  [oai_citation:11‡ВычГем.pdf](sediment://file_000000001e30722f9b0d1b0dfaa8478b)

### Файлы решения
- `lab4/task_4_1_convex_hull.ipynb`
- `lab4/task_4_2_voronoi_delaunay.ipynb`
